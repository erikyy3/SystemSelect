# SystemSelect
Status do Projeto: Em desenvolvimento
Tecnologias que serão aplicadas: Java e Mysql
Time de desenvolvedores: Erick Brandenburg Holtz da Cruz
Objetivo do software: O objetivo do nosso sistema é auxiliar e simplificar a administração das lojas de computadores como também para o consumidor, podendo consultar suas compras e administrar o seus gastos.
Funcionalidades do sistema: Administração de Vendas(Inserção, Exclusão e Edição, Administração de Estoque(Inserção, Exclusão e Edição), Administração de Usuários(Inserção, Exclusão e Edição), Consulta de Compras do cliente e Alteração de dados do Cliente.
